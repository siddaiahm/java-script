Rules to be followed for SUDUKU Game.

  • The Classic Suduku is a number placing puzzle based on a 9X9 grid with several given numbers.
  • The object is to place the number from 1 to 9 in the empty squares so that each row, each column and each 3X3 box contains the same number only once.
